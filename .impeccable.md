# podkit — design context

## Design Context

### Users
Developers and SREs operating apps on podkit — and AI agents acting alongside them. Context: long sessions in dark editors/terminals; they come to the dashboard to deploy, watch, diagnose, and manage data/auth. Power users who value density and speed over hand-holding.

### Brand Personality
Precise, capable, calm-technical. Three words: **precise · capable · unfussy**. The interface should feel like a well-made instrument — confident, legible, never decorative for its own sake. Quiet refinement inside high information density.

### Aesthetic Direction
- **Theme: dark.** Tinted near-black surfaces (warm-neutral, tinted toward the brand hue) — never pure black, never pure white.
- **Accent: a single distinctive amber/ochre** (warm "signal" color), used rarely (active nav, primary action, key status). Status semantics: green = healthy/promoted, amber = pending/attention, red = error/rollback. NOT neon, NOT cyan-on-dark, NOT purple gradients.
- **Density first** — tables and lists over card grids; this is a console, not a marketing page. Refinement shows in spacing rhythm, type, and restraint, not whitespace bloat.
- **Type:** UI = **Hanken Grotesk**; values/IDs/code = **Spline Sans Mono** (versionIds, deployIds, tokens, timestamps, routes set in mono). Hierarchy via weight/size, not a display font.
- **Agents-first cues:** show identity (human vs agent) where relevant; mono/`--json`-parity feel; machine-legible.

### Anti-references (avoid)
Generic AI dashboards: cyan/purple neon on black, glassmorphism, identical icon+heading+text card grids, big-number hero metric blocks, gradient text, side-stripe colored borders, rounded-rect drop-shadow cards everywhere.

### Design Principles
1. **Instrument, not interface** — every element earns its place; legibility and speed first.
2. **Density with rhythm** — pack information, but vary spacing so hierarchy reads.
3. **Mono for machine truth** — IDs, versions, timestamps, routes, tokens in monospace.
4. **Rare accent** — amber appears only where it means something.
5. **No AI tells** — if it looks like default AI output, redo it.
